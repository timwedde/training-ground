# training-ground

## Usage

### Install `uv`

```bash
command -v uv || curl -LsSf https://astral.sh/uv/install.sh | sh
```

### Install the Tool

```bash
uv tool install --from git+https://github.com/timwedde/training-ground.git training-ground
```

### Run the Tool

```bash
training-ground --help
```
